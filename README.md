## 👤 About

Personal portfolio of **Anup Sahu** — Cybersecurity Analyst (VAPT) & Full-Stack Developer based in Bengaluru, India. This site showcases my skills, projects, security stats, and experience in penetration testing and web development.

---

## ✨ Features

- ⚡ Animated particle background & glitch effects
- 🎯 Typewriter role animation
- 📊 Live LeetCode & PortSwigger security stats
- 🌐 Responsive design — mobile & desktop
- 📬 Contact form via Formspree
- 📄 Downloadable resume
- 🔍 Google Analytics (via GTM)
- 🌑 Full dark cybersecurity theme

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/SuperANUP07/PortFolio.git

# Navigate to the portfolio folder
cd PortFolio/portfolio

# Open in browser
open index.html
# or just drag index.html into your browser
```

No build step needed — it's pure HTML/CSS/JS. ✅

---

## ✏️ Customize

All content is in one file — **`portfolio/config.js`**. Edit it to update:

- Your name, roles, description
- Skills and tech stack
- Projects and live links
- Experience and certifications
- Security stats
- Contact links
